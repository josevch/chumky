# Tailwind CSS in WordPress This repository is dedicated to testing and implementing Tailwind CSS in WordPress projects. ## Usage 1. Modify the `tailwind.config.js` file to customize Tailwind. 2. Write your CSS using Tailwind's utility classes. 3. Compile your CSS using `npm run build`. 4. Link the compiled CSS file in your WordPress project. ## Contributing Contributions are welcome!
